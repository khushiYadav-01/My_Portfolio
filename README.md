# Khushi Yadav — AI Engineer Portfolio

A personal portfolio site built with a **React (Vite) frontend** and a **Node.js / Express backend**. Projects are stored and served through a REST API rather than hardcoded in the UI, so new work can be added without touching any component code.

---

## ✨ Features

- Responsive, animated single-page design (dark theme, scroll reveals, orbiting skill chips)
- Fixed header nav with a mobile hamburger menu
- "About" section with an animated training-loss-curve timeline of career milestones
- **Projects section backed by a real API** — filterable by category (NLP/LLMs, Computer Vision, Data/MLOps), scales to any number of entries
- Working contact form that submits to the backend and stores messages
- Résumé section with video + downloadable PDF résumé slots
